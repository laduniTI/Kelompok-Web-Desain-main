Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman bernama "/action_page.php".

Kesimpulan Dari Program Diatas Adalah jika kita menggunakan method post,maka data dikirim secara langsung. Jika Kita memilih Kirim Maka semua data langsung di kirim ke file tujuan,sedangkan jika kita pilih reset maka semua data akan terhapus
